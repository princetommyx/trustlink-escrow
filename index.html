The Cross-Platform Transaction Engine

Because transactions happen across fragmented apps, we act as the absolute source of truth.

1

Initiated

Seller configures the payment parameters. A secure unique token reference is minted and shared.

2

Awaiting Payment

Buyer authenticates the Moolre push notification request on their mobile device via the web view.

3

Funds Escrowed

Moolre webhook received. Money is ring-fenced. Seller receives a WhatsApp alert to dispatch.

4

Item Shipped

Seller hits "Mark as Shipped." This fires the 72-hour auto-release background countdown daemon.

5

Completed

Buyer taps "Confirm Receipt" or 72h timer elapses. Moolre Disbursements API triggers vendor payout.
